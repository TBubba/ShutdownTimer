Timer

The timer both the place where you input the desired time before shutdown. This is also where the countdown will occur when started.

Format

Input: The timer only accepts numbers (0-9) and colons (":").
Format: The timer only allows the following formats: hh:mm:ss mm:ss ss
              Note: Leading zeros are not necessary / optional

Shutdown Mode

Shutdown modes are different actions that can be taken once the countdown is complete. Only the selected one will be executed upon countdown completion.

Modes

There are currently 6 modes to choose from. They are the following;

  • Shutdown,
  • Restart,
  • Hibernate (Power saving mode),
  • Sleep,
  • Log out and
  • Lock

Time Mode

Time modes are ways for the timer to count down.

Modes

There are currently 2 modes to choose from. They are the following;

  • Countdown - counts down from inputted time to zero
  • Wait until - counts down from current time to inputted time

Start/Stop

The Start/Stop button is very straight forward. When pressing the start button, the timer starts counting down. And when pressing the stop button, the timer stops.
